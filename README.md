# r_get_ip_v3
- Imp mere clone of r_get_ip to make it flutter 3 compitable
[![pub package](https://img.shields.io/pub/v/r_get_ip.svg)](https://pub.dartlang.org/packages/r_get_ip)

![](screen/1.jpg)

A new Flutter project about get ip address.

# Supported Platform

- Android
- IOS
- MacOS
- Web
- Windows

# Getting Started

1. Get Internal IP Address.

```dart
RGetIp.internalIP
```

2. Get External IP
```dart
RGetIp.externalIP
```

3. Get Network Type
```dart
RGetIp.networkType
```
